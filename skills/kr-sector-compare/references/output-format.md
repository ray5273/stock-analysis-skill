# Output Format

## Default Compare Memo

1. Bottom Line
   State which sector looks stronger and why.
2. Comparison Frame
   State date, scope, and exclusions.
3. Same-Date Snapshot Table
   Use a compact table for the few metrics that matter.
4. Where Sector A Is Stronger
5. Where Sector B Is Stronger
6. Listed Exposure Quality
7. Key Risks and What Could Change the Ranking
8. Sources

## Tone

- Be decisive when the evidence is clear.
- Be explicit when the evidence is mixed.
