# Workflow Reference

## Goal

Compare sectors on a common frame without forcing them into fake precision.

## Comparison Axes

Use only the axes that matter for the pair:

- market structure
- demand visibility
- policy support
- regulatory drag
- capital intensity
- listed pure-play availability
- earnings sensitivity
- near-term event path

## Sequence

1. Normalize the pair.
2. Choose 3-6 comparison axes.
3. Gather same-date facts.
4. Explain structural differences.
5. Rank only when the source set supports it.

## Failure Modes To Avoid

- comparing mixed-date inputs
- pretending sector metrics are interchangeable when they are not
- burying the bottom line under too many side details
