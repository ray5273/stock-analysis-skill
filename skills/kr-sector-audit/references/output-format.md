# Output Format

## Default Audit Memo

1. Findings
   List issues in severity order with section references when possible.
2. Open Questions
   Note ambiguities that need source checks or scope decisions.
3. Residual Risks
   State what may still be fragile even if there are few findings.
4. Change Summary
   Keep this brief and secondary to the findings.

## Finding Style

Use concise labels such as:

- `[High] Korea-only market size is unsupported`
- `[Medium] Mixed-date metrics presented as one snapshot`
- `[Medium] Listed exposure map overstates direct exposure`
- `[Low] Missing explicit source date in policy section`

## Tone

- Be direct.
- Focus on correctness and decision risk.
