# Output Format

## Default Memo Header

Keep or add:

```md
기준일: 2026-03-29
최근 업데이트일: 2026-04-10
```

`기준일` is the original memo date.  
`최근 업데이트일` is the most recent follow-up refresh date.

## Default Update Log Shape

Append updates to the existing memo in this form:

```md
## Update Log

### 2026-04-10 Update

#### What changed
- ...

#### Why it matters
- ...

#### What did not change
- ...

#### What to watch next
- ...

#### Sources
- [Source label](https://example.com)
```

## Tone

- Be direct.
- Keep the update tied to the base thesis.
- Prefer a short dated block over a full rewrite.
